# What is Centuply

Centuply is a platform for creating professional websites with the help of AI.

It helps you create a complete website for your business, edit it easily, and publish it when you are ready.

You do not need to start from a blank page. Centuply guides you step by step and creates an initial structure, content, and sections based on your business needs.

## Who it is for

Centuply is useful for small and medium businesses, professionals, and teams that want a serious website quickly.

Examples:

- restaurants,
- cafés,
- clinics,
- hair salons,
- beauty salons,
- technical services,
- consultants,
- trainers,
- freelancers,
- small shops,
- local businesses.

## What you can create

With Centuply, you can create websites such as:

- business website,
- landing page,
- service website,
- local business site,
- portfolio,
- appointment-based website,
- promotional site,
- multilingual website,
- website with forms and leads,
- website with blog or content.

## How it works

The basic flow is simple:

1. Describe your business.
2. Choose basic preferences.
3. AI creates an initial website proposal.
4. Review the result.
5. Edit text, images, and sections.
6. Add forms, SEO, domain, and other settings.
7. Publish when you are ready.

## What AI does

AI helps create the first draft.

It can suggest:

- page structure,
- headlines,
- descriptions,
- sections,
- calls to action,
- service copy,
- brand tone,
- basic content,
- improvement ideas.

AI does not lock you into one result. You can edit and change anything you want.

## Why it is useful

Centuply is useful because it reduces the time needed to start a professional website.

Instead of thinking from scratch about:

- which pages you need,
- what text to write,
- how to organize services,
- where to place CTAs,
- how to present the business,

Centuply gives you an organized starting point.

## What you can edit

Inside the editor, you can change:

- headlines,
- subtitles,
- text,
- images,
- buttons,
- sections,
- pages,
- colors,
- tone,
- forms,
- SEO details,
- domain and publish settings.

The goal is to give you control over the final result.

## Centuply Assistant

Centuply Assistant helps you inside the editor.

You can ask for ideas or changes, such as:

- improve the headline,
- make the text shorter,
- make the section more premium,
- suggest a better CTA,
- make the page cleaner,
- improve the tone.

Changes are shown first as a preview, so you can review them before applying.

## Preview before changes

Centuply focuses on control.

When an AI suggestion changes content or appearance, you should see it as a preview first.

This lets you decide:

- whether you like it,
- whether it matches the brand,
- whether you want to apply it,
- whether you want to adjust it.

## Website publishing

The website does not need to be published immediately.

You can work in draft, make changes, and publish only when you are ready.

Before publishing, it is useful to check:

- content,
- mobile appearance,
- forms,
- links,
- SEO,
- domain,
- contact details.

## What Centuply is not

Centuply is not just a template library.

It does not only give you a ready layout to fill in manually.

It combines AI generation, editor, assistant, pages, sections, forms, SEO, publishing, and business tools so you have a more complete website creation flow.

## Best practices for new users

- Start with a simple business description.
- Do not try to make everything perfect immediately.
- Check the general structure first.
- Then improve copy and images.
- Test the website on mobile.
- Add clear CTAs.
- Check forms and contact details.
- Publish only when you are confident.

## Frequently asked questions

### Do I need to know programming?

No. Centuply is designed so you can create and edit a website without writing code.

### Does AI publish the website by itself?

No. You review the result and decide when to publish.

### Can I change the website after generation?

Yes. You can edit text, images, sections, pages, and settings.

### Is it only for one type of business?

No. It can be used by many types of small businesses and professionals.

### Can I use my own domain?

Yes. You can connect a custom domain when you are ready to publish.
