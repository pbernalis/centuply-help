# AI Builder overview

The AI Builder helps you create a professional website from a short business description.

It is the starting point for generating a first website draft that you can later review, edit, improve, and publish.

## What the AI Builder does

The AI Builder can help create:

- initial page structure,
- homepage sections,
- service descriptions,
- calls to action,
- business copy,
- contact areas,
- SEO-friendly starting content,
- a first visual direction.

The first result is not final. It is a starting point that you can edit inside Centuply.

## Main creation flows

Centuply may offer more than one creation flow.

### Simple Mode

Simple Mode is the fastest guided setup.

Use it when you want to start quickly and provide only the basic information.

### Advanced Planner

Advanced Planner gives you more control before generation.

Use it when you want to define services, audience, pages, goals, tone, and features in more detail.

## After generation

After the website is generated, you can open the editor and change:

- text,
- images,
- sections,
- pages,
- buttons,
- forms,
- SEO settings,
- publishing settings.

You can also use the Centuply Assistant to suggest improvements and preview changes before applying them.

## Best practices

- Describe your business clearly.
- Add your main services.
- Choose the right website goal.
- Review the generated draft before publishing.
- Use the editor to improve the first result.
- Check mobile view, forms, and SEO before going live.
