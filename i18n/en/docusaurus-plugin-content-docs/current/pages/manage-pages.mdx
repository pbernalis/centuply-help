# Pages Manager: manage pages

Pages Manager is where you organize the pages of your website.

From here, you can see which pages exist, add new pages, reorder them, set the home page, hide pages from navigation, or manage the overall website structure.

## When to use Pages Manager

Use Pages Manager when you want to:

- see all pages of the website,
- add a new page,
- duplicate an existing page,
- rename a page,
- change page order,
- hide or show a page in the menu,
- set which page is the homepage,
- organize the structure before publishing.

## What is a page?

A page is a main part of your website.

Examples:

- Home
- About
- Services
- Pricing
- Gallery
- Blog
- Contact
- FAQ
- Privacy Policy
- Terms

Each page can have its own sections, text, images, and SEO settings.

## Step 1 — Open Pages Manager

Open the Centuply app, select your website, and open **Pages** or **Pages Manager**.

You will see the list of pages already included in the website.

## Step 2 — Review the page list

Look at the existing pages and check whether the structure makes sense for visitors.

Check:

- if there is a Home page,
- if there is a Contact page,
- if services are presented clearly,
- if there are unnecessary pages,
- if an important page is missing,
- if page names are easy to understand.

## Step 3 — Add a new page

To add a page, click **Add page** or **Create page**.

Common pages you may need:

- Services
- About
- Contact
- Pricing
- Gallery
- FAQ
- Blog
- Booking
- Locations
- Legal pages

Give the page a clear name that visitors can easily understand.

## Step 4 — Duplicate an existing page

If you want to create a similar page, you can use duplicate.

Examples:

- one page for each service,
- one page for each location,
- one page for each product,
- one page for each case study.

After duplicating, update the titles, text, images, and SEO so the new page is unique.

## Step 5 — Change page order

Page order affects the navigation menu.

A simple order for many business websites is:

1. Home
2. Services
3. About
4. Gallery or Blog
5. Contact

Place the most helpful pages first.

## Step 6 — Hide or show a page in the menu

Not every page needs to appear in the main menu.

You can hide pages such as:

- Privacy Policy,
- Terms,
- special landing pages,
- temporary pages,
- pages that are not ready yet.

A hidden page can still exist on the website without appearing in navigation.

## Step 7 — Set the homepage

The homepage is the first page visitors see.

It should clearly explain:

- who the business is,
- what it offers,
- why visitors should trust it,
- what visitors should do next.

If you change the homepage, carefully check navigation and links.

## Step 8 — Use recommended pages

Centuply may suggest pages that are missing from the website.

Examples:

- Contact page,
- Services page,
- FAQ page,
- Privacy Policy,
- Booking page,
- Blog.

If a recommended page fits the business, you can add it.

## Step 9 — Review the structure like a visitor

Before publishing, review the structure as if you were a customer.

Ask:

- Can I quickly understand what the business offers?
- Can I easily find the services?
- Can I easily find contact details?
- Are there too many unnecessary pages?
- Is the menu simple?
- Are legal pages included if needed?

## Best practices

- Keep the menu simple.
- Do not add too many pages without a reason.
- Use clear page names.
- Keep the Contact page easy to access.
- Do not leave empty pages.
- Hide pages that are not ready.
- Always check mobile navigation.

## Frequently asked questions

### How many pages does a small business website need?

Usually Home, Services, About, and Contact are enough. If needed, you can add Gallery, FAQ, Blog, or Booking.

### Can I hide a page without deleting it?

Yes. You can remove it from navigation and keep it available as a separate page.

### What does duplicate page mean?

It creates a copy of an existing page so you can customize it quickly.

### Should all pages appear in the menu?

No. The menu should only include the main pages that help visitors.

### What should I do before deleting a page?

Check whether the page contains important content, links, SEO value, or is linked from other parts of the website.
