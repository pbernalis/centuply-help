# Resumen de AI Builder

AI Builder te ayuda a crear un sitio web profesional a partir de una breve descripción del negocio.

Es el punto de partida para generar un primer draft del sitio, que después puedes revisar, editar, mejorar y publicar.

## Qué hace AI Builder

AI Builder puede ayudar a crear:

- estructura inicial de páginas,
- secciones de la homepage,
- descripciones de servicios,
- llamadas a la acción,
- textos del negocio,
- áreas de contacto,
- contenido inicial orientado a SEO,
- una primera dirección visual.

El primer resultado no es definitivo. Es una base inicial que puedes editar dentro de Centuply.

## Flujos principales de creación

Centuply puede ofrecer más de un flujo de creación.

### Simple Mode

Simple Mode es la configuración guiada más rápida.

Úsalo cuando quieras empezar rápido y aportar solo la información básica.

### Advanced Planner

Advanced Planner ofrece más control antes de la generación.

Úsalo cuando quieras definir servicios, audiencia, páginas, objetivos, tono y funciones con más detalle.

## Después de la generación

Después de generar el sitio, puedes abrir el editor y cambiar:

- textos,
- imágenes,
- secciones,
- páginas,
- botones,
- formularios,
- ajustes SEO,
- ajustes de publicación.

También puedes usar Centuply Assistant para sugerir mejoras y ver preview antes de aplicar cambios.

## Buenas prácticas

- Describe tu negocio con claridad.
- Añade los servicios principales.
- Elige el objetivo correcto del sitio.
- Revisa el draft generado antes de publicar.
- Usa el editor para mejorar el primer resultado.
- Revisa móvil, formularios y SEO antes de hacerlo live.
