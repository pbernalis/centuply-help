# Simple Mode: crear un sitio con configuración guiada

Simple Mode es el flujo más fácil para crear un sitio web en Centuply.

Te guía paso a paso y pide solo la información básica que la AI necesita para crear una primera propuesta profesional de sitio web.

## Cuándo usar Simple Mode

Usa Simple Mode cuando:

- quieras empezar rápido,
- no quieras muchas opciones técnicas,
- quieras una guía simple,
- aún no sepas la estructura exacta del sitio,
- quieras que la AI sugiera contenido inicial,
- estés creando un sitio para un pequeño negocio,
- quieras editar después dentro del editor.

## Qué pide Simple Mode

Normalmente pide datos básicos como:

- tipo de negocio,
- nombre del negocio,
- descripción breve,
- servicios,
- ubicación,
- idioma,
- tono,
- objetivo principal del sitio,
- datos de contacto,
- funciones opcionales.

No necesitas textos perfectos. Una descripción clara es suficiente.

## Paso 1 — Abre Create Website

Abre la aplicación de Centuply y selecciona **Create website** o **New site**.

Si aparecen opciones de creación, elige **Simple Mode** o el flujo guiado.

## Paso 2 — Describe tu negocio

Describe tu negocio con palabras simples.

Ejemplos:

- “Clínica dermatológica en Piraeus”
- “Peluquería para mujeres y hombres”
- “Restaurante con cocina mediterránea”
- “Técnico de ordenadores con visitas a domicilio”
- “Consultor de negocios para pequeñas empresas”

Cuanto más claro expliques lo que haces, mejor será la propuesta inicial.

## Paso 3 — Añade servicios

Escribe los servicios principales que ofreces.

Ejemplo para peluquería:

- Corte,
- Color,
- Peinado,
- Tratamientos capilares,
- Peinado de novia.

No añadas demasiados detalles al principio. Puedes añadirlos después.

## Paso 4 — Elige objetivo del sitio

El objetivo ayuda a la AI a organizar mejor el sitio.

Ejemplos de objetivos:

- generar leads,
- aumentar reservas,
- presentar servicios,
- dar información,
- fortalecer la marca,
- ayudar a los visitantes a contactar.

Si el objetivo principal es que los clientes llamen, el sitio debe tener CTAs de contacto fuertes.

## Paso 5 — Añade ubicación y contacto

Si el negocio es local, añade la ubicación.

Ejemplos:

- Piraeus,
- Athens,
- Glyfada,
- Thessaloniki,
- servicios online,
- servicio nacional.

Añade también teléfono, email o dirección si están disponibles.

## Paso 6 — Elige tono

El tono afecta los textos y la sensación del sitio.

Ejemplos:

- profesional,
- premium,
- cercano,
- moderno,
- minimal,
- serio,
- cálido,
- dinámico.

Por ejemplo, una clínica necesita un tono más confiable y limpio, mientras que un café puede usar un tono más cálido y cercano.

## Paso 7 — Revisa el resumen

Antes de continuar, revisa el resumen de información.

Asegúrate de que:

- el tipo de negocio sea correcto,
- los servicios sean correctos,
- la ubicación sea correcta,
- el idioma sea correcto,
- el objetivo del sitio sea correcto,
- no haya errores en teléfono o email.

## Paso 8 — Continúa a generación

Cuando los datos sean correctos, continúa con la generación del sitio.

Centuply usará la información para crear el primer draft.

La generación puede incluir:

- estructura inicial de páginas,
- secciones,
- titulares,
- descripciones,
- CTAs,
- servicios,
- áreas de contacto,
- tono sugerido.

## Paso 9 — Revisa el resultado

Cuando la generación termine, revisa el sitio cuidadosamente.

Comprueba:

- si el hero explica bien el negocio,
- si los servicios son correctos,
- si el tono encaja,
- si los CTAs son claros,
- si la estructura tiene lógica,
- si falta información importante.

## Paso 10 — Continúa en el editor

Simple Mode crea la base inicial.

Después, dentro del editor puedes cambiar:

- textos,
- imágenes,
- secciones,
- páginas,
- botones,
- colores,
- formularios,
- SEO,
- ajustes de publicación y dominio.

La primera propuesta de AI no tiene que ser perfecta. Es el punto de partida.

## Buenas prácticas

- Escribe una descripción clara del negocio.
- Evita frases genéricas sin significado.
- Añade los servicios principales.
- Elige el objetivo correcto del sitio.
- Revisa los datos antes de generar.
- No intentes escribir todo perfecto al principio.
- Usa el editor para mejorar el sitio después de la generación.

## Preguntas frecuentes

### ¿Simple Mode es para principiantes?

Sí. Es el flujo más simple para empezar sin muchas configuraciones.

### ¿Puedo cambiar el sitio después?

Sí. Después de la generación puedes editar el sitio dentro del editor.

### ¿Necesito tener todos los textos listos?

No. La AI puede crear textos iniciales según tu descripción.

### ¿Qué pasa si el resultado no es perfecto?

Úsalo como base y mejóralo con el editor y Centuply Assistant.

### ¿Cuándo no debería usar Simple Mode?

Si quieres control muy detallado desde el inicio, puede que prefieras un flujo más avanzado.
