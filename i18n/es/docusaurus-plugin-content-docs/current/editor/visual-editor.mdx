# Conceptos básicos del Visual Editor

El Visual Editor es el espacio principal donde editas tu sitio web después de que se ha creado con IA.

Dentro del editor puedes cambiar textos, imágenes, secciones, colores, diseño y elementos básicos de la página sin escribir código.

## Cuándo usar el Visual Editor

Usa el Visual Editor cuando quieras:

- cambiar titulares y textos,
- añadir o eliminar secciones,
- cambiar imágenes,
- revisar la apariencia del sitio,
- ver cómo se ve el sitio en desktop y móvil,
- hacer pequeñas mejoras antes de publicar,
- revisar cambios sugeridos por el AI Assistant.

## Paso 1 — Abre tu sitio web

Abre la aplicación de Centuply y selecciona el sitio que quieres editar.

Desde el dashboard o Site Cockpit, haz clic en **Edit** o **Open Editor**.

## Paso 2 — Entiende la estructura básica

El Visual Editor normalmente tiene varias áreas principales:

- **Área de vista previa**: muestra la página como la verán los visitantes.
- **Sidebar**: contiene herramientas para secciones, diseño, layout, historial e IA.
- **Inspector**: permite cambiar propiedades de la sección o elemento seleccionado.
- **Barra superior**: incluye opciones de preview, publish, save o navegación.

La apariencia exacta puede variar según el tamaño de la pantalla y el modo que estés usando.

## Paso 3 — Selecciona una sección

Haz clic en una sección de la página, como:

- Header
- Hero
- Services
- About
- Gallery
- Testimonials
- Contact
- Footer

Cuando seleccionas una sección, el editor muestra las opciones disponibles para esa parte específica de la página.

## Paso 4 — Edita textos

Para editar textos, selecciona la sección o el elemento que contiene el texto.

Puedes cambiar:

- titular,
- subtítulo,
- párrafo,
- texto del botón,
- etiquetas,
- textos de ayuda pequeños.

Antes de guardar, lee el resultado como si fueras un visitante. El texto debe ser claro, breve y útil.

## Paso 5 — Cambia imágenes

Si una sección admite una imagen, puedes cambiarla desde las opciones del editor o desde la Media Library.

Usa imágenes que:

- muestren claramente el negocio,
- encajen con el estilo del sitio,
- no estén borrosas,
- se vean bien en móvil,
- no sean demasiado pesadas.

## Paso 6 — Añade o cambia secciones

Desde el panel de secciones puedes añadir nuevas secciones a la página.

Ejemplos de secciones:

- Services
- Pricing
- Gallery
- Testimonials
- FAQ
- CTA Banner
- Contact
- Map

Añade solo secciones que ayuden al visitante a entender el negocio o realizar una acción, como contactar, reservar o comprar.

## Paso 7 — Usa el AI Assistant

Si no sabes cómo mejorar una sección, usa el AI Assistant.

Ejemplos de prompts:

> Haz que el hero se vea más premium.

> Haz el texto más corto.

> Sugiere un mejor CTA.

> Haz que esta sección sea más adecuada para reservas.

El Assistant muestra una vista previa antes de aplicar cambios.

## Paso 8 — Revisa desktop y móvil

Antes de publicar, revisa cómo se ve la página en diferentes tamaños de pantalla.

Presta atención a:

- si los titulares encajan correctamente,
- si los botones se ven claros,
- si las imágenes se recortan de forma extraña,
- si las secciones tienen buen espaciado,
- si la página es fácil de leer en móvil.

## Paso 9 — Guarda los cambios

Cuando termines de editar, guarda los cambios.

Si el sitio no está listo para publicarse, mantenlo como draft y continúa más tarde.

Si está listo, continúa con el flujo de publicación.

## Buenas prácticas

- No cambies demasiadas cosas a la vez.
- Haz pequeños cambios y revisa la vista previa.
- Mantén los textos breves.
- Usa imágenes claras.
- No sobrecargues la página con demasiadas secciones.
- Revisa siempre la vista móvil.
- Usa el AI Assistant para obtener sugerencias, pero mantén el control final.

## Preguntas frecuentes

### ¿Necesito código para usar el Visual Editor?

No. El Visual Editor está diseñado para editar sin código.

### ¿Puedo cambiar todo el sitio desde el editor?

Puedes cambiar las páginas principales, secciones, textos, imágenes y apariencia. Para cambios más avanzados, usa áreas como Brand, SEO, Forms o Domains.

### ¿Qué pasa si cometo un error?

Puedes no aplicar una propuesta, corregir el contenido o continuar con un nuevo cambio. Lo mejor es revisar la vista previa antes de avanzar.

### ¿Tengo que publicar después de cada cambio?

No. Puedes editar el sitio y publicar cuando esté listo.

### ¿Cuándo debo usar el AI Assistant en lugar de editar el texto manualmente?

Úsalo cuando quieras una mejor redacción, un tono más profesional, un CTA más fuerte o ideas de mejora.
