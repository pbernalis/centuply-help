# Media Library: imágenes, archivos y organización

La Media Library es el lugar donde gestionas las imágenes y los assets multimedia de tu sitio web.

Desde aquí puedes subir fotos, organizar archivos, reutilizar imágenes en diferentes páginas y mantener ordenado el contenido del sitio.

## Cuándo usar la Media Library

Usa la Media Library cuando quieras:

- subir imágenes del negocio,
- cambiar imágenes en secciones,
- organizar fotos en folders,
- encontrar imágenes ya subidas,
- reutilizar imágenes en otra página,
- comprobar qué assets se están usando,
- limpiar media antiguo o no utilizado.

## Qué puedes guardar

Según la configuración de tu sitio, la Media Library puede incluir:

- imágenes,
- archivos de logo,
- iconos,
- videos o enlaces de video,
- documentos,
- assets generados,
- URLs externas de media.

Las imágenes suelen ser los assets más importantes para un sitio web profesional de negocio.

## Paso 1 — Abre la Media Library

Abre la aplicación de Centuply, selecciona tu sitio y abre **Media** o **Media Library**.

Verás los assets ya subidos o usados en el sitio.

## Paso 2 — Sube una nueva imagen

Haz clic en **Upload** o arrastra imágenes dentro de la Media Library.

Antes de subirlas, comprueba que las imágenes:

- sean claras,
- tengan buena resolución,
- no sean demasiado oscuras,
- no sean demasiado grandes sin motivo,
- encajen con el estilo del sitio.

## Paso 3 — Organiza los media

Si tienes muchas imágenes, organízalas con folders o tags.

Ejemplos de organización:

- Hero images
- Services
- Team
- Products
- Gallery
- Logo
- Backgrounds
- Social

Una buena organización ayuda mucho cuando el sitio crece.

## Paso 4 — Usa una imagen en una sección

Cuando editas una sección en el Visual Editor, puedes seleccionar una imagen desde la Media Library.

Ejemplos:

- imagen hero en la página de inicio,
- imagen de servicio,
- foto del equipo,
- imagen de gallery,
- imagen de fondo,
- logo en el header.

## Paso 5 — Comprueba dónde se usa un asset

Antes de eliminar una imagen, comprueba si se usa en alguna página o sección.

Si eliminas una imagen que se está usando, la página puede dejar de verse correctamente.

Es mejor reemplazar primero la imagen y eliminarla después.

## Paso 6 — Limpia assets antiguos o no utilizados

Con el tiempo puedes acumular imágenes que ya no necesitas.

Puedes limpiar:

- imágenes antiguas,
- duplicados,
- uploads temporales,
- imágenes no utilizadas,
- assets que ya no encajan con la marca.

No elimines archivos demasiado rápido si no sabes dónde se usan.

## Paso 7 — Revisa la apariencia móvil

Una imagen puede verse bien en desktop pero no en móvil.

Después de cambiar una imagen, revisa:

- si se recorta correctamente,
- si el rostro o producto se ve,
- si el texto sigue siendo legible,
- si la sección se ve equilibrada.

## Buenas prácticas para imágenes

- Prefiere fotos reales del negocio.
- Usa imágenes claras y luminosas.
- Evita fotos borrosas.
- No subas muchas versiones de la misma imagen sin motivo.
- Mantén un estilo coherente.
- Revisa siempre cómo se ve la imagen en móvil.
- No uses imágenes sin derechos de uso.

## Preguntas frecuentes

### ¿Puedo subir mis propias fotos?

Sí. Puedes subir fotos desde tu ordenador y usarlas en el sitio.

### ¿Tengo que organizar las imágenes en folders?

No es obligatorio, pero ayuda mucho cuando tienes muchos assets.

### ¿Puedo usar la misma imagen en varias páginas?

Sí. Una imagen puede usarse en varios lugares.

### ¿Qué pasa si elimino una imagen que se está usando?

Puede desaparecer de la página o afectar la apariencia. Comprueba primero dónde se usa.

### ¿Qué imágenes debo preferir?

Usa imágenes reales, claras y luminosas que muestren el negocio, el equipo, el espacio o los productos.
