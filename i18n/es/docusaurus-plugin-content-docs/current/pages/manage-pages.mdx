# Pages Manager: gestionar páginas

Pages Manager es el lugar donde organizas las páginas de tu sitio web.

Desde aquí puedes ver qué páginas existen, añadir nuevas páginas, cambiar el orden, definir la página de inicio, ocultar páginas del menú o gestionar la estructura general del sitio.

## Cuándo usar Pages Manager

Usa Pages Manager cuando quieras:

- ver todas las páginas del sitio,
- añadir una nueva página,
- duplicar una página existente,
- renombrar una página,
- cambiar el orden de las páginas,
- ocultar o mostrar una página en el menú,
- definir qué página es la página de inicio,
- organizar mejor la estructura antes de publicar.

## Qué es una página

Una página es una parte principal de tu sitio web.

Ejemplos:

- Home
- About
- Services
- Pricing
- Gallery
- Blog
- Contact
- FAQ
- Privacy Policy
- Terms

Cada página puede tener sus propias secciones, textos, imágenes y ajustes SEO.

## Paso 1 — Abre Pages Manager

Abre la aplicación de Centuply, selecciona tu sitio y abre **Pages** o **Pages Manager**.

Verás la lista de páginas que ya existen en el sitio.

## Paso 2 — Revisa la lista de páginas

Observa las páginas existentes y comprueba si la estructura tiene sentido para el visitante.

Revisa:

- si existe una página Home,
- si existe una página Contact,
- si los servicios se presentan claramente,
- si hay páginas innecesarias,
- si falta una página importante,
- si los nombres son fáciles de entender.

## Paso 3 — Añade una nueva página

Para añadir una página, haz clic en **Add page** o **Create page**.

Páginas comunes que puedes necesitar:

- Services
- About
- Contact
- Pricing
- Gallery
- FAQ
- Blog
- Booking
- Locations
- Legal pages

Da a la página un nombre claro que el visitante pueda entender fácilmente.

## Paso 4 — Duplica una página existente

Si quieres crear una página parecida, puedes usar duplicate.

Ejemplos:

- una página para cada servicio,
- una página para cada zona,
- una página para cada producto,
- una página para cada caso de estudio.

Después de duplicarla, cambia títulos, textos, imágenes y SEO para que la nueva página sea única.

## Paso 5 — Cambia el orden de las páginas

El orden de las páginas afecta al menú de navegación.

Un orden simple para muchos sitios de negocio es:

1. Home
2. Services
3. About
4. Gallery o Blog
5. Contact

Coloca primero las páginas que más ayudan al visitante.

## Paso 6 — Oculta o muestra una página en el menú

No todas las páginas tienen que aparecer en el menú principal.

Puedes ocultar páginas como:

- Privacy Policy,
- Terms,
- landing pages especiales,
- páginas temporales,
- páginas que aún no están listas.

Una página oculta puede seguir existiendo en el sitio sin aparecer en la navegación.

## Paso 7 — Define la página de inicio

La página de inicio es la primera página que ve el visitante.

Debe explicar claramente:

- quién es el negocio,
- qué ofrece,
- por qué el visitante debería confiar,
- qué debe hacer después.

Si cambias la página de inicio, revisa bien la navegación y los enlaces.

## Paso 8 — Usa páginas recomendadas

Centuply puede sugerir páginas que faltan en el sitio.

Ejemplos:

- Contact page,
- Services page,
- FAQ page,
- Privacy Policy,
- Booking page,
- Blog.

Si una página recomendada encaja con el negocio, puedes añadirla.

## Paso 9 — Revisa la estructura como visitante

Antes de publicar, revisa la estructura como si fueras un cliente.

Pregúntate:

- ¿Puedo entender rápidamente qué ofrece el negocio?
- ¿Encuentro fácilmente los servicios?
- ¿Encuentro fácilmente los datos de contacto?
- ¿Hay demasiadas páginas innecesarias?
- ¿El menú es simple?
- ¿Hay páginas legales si hacen falta?

## Buenas prácticas

- Mantén el menú simple.
- No añadas demasiadas páginas sin motivo.
- Usa nombres claros.
- Mantén la página Contact fácil de encontrar.
- No dejes páginas vacías.
- Oculta páginas que no están listas.
- Revisa siempre la navegación móvil.

## Preguntas frecuentes

### ¿Cuántas páginas necesita un sitio pequeño?

Normalmente Home, Services, About y Contact son suficientes. Si hace falta, puedes añadir Gallery, FAQ, Blog o Booking.

### ¿Puedo ocultar una página sin eliminarla?

Sí. Puedes quitarla de la navegación y mantenerla disponible como página separada.

### ¿Qué significa duplicar página?

Crea una copia de una página existente para que puedas personalizarla rápidamente.

### ¿Todas las páginas deben aparecer en el menú?

No. En el menú deben aparecer solo las páginas principales que ayudan al visitante.

### ¿Qué debo hacer antes de eliminar una página?

Revisa si la página tiene contenido importante, enlaces, valor SEO o si está enlazada desde otras partes del sitio.
