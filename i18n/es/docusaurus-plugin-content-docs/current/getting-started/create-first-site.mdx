# Crea tu primer sitio web con IA

En esta guía aprenderás cómo crear tu primer sitio web con Centuply, sin necesitar conocimientos técnicos ni experiencia en diseño.

Centuply te ayuda a describir tu negocio con palabras simples y te entrega un sitio web inicial que luego puedes editar, mejorar y publicar.

## Cuándo usar esta guía

Usa esta guía cuando:

- estás usando Centuply por primera vez,
- quieres crear un nuevo sitio web,
- no sabes por dónde empezar,
- quieres obtener rápidamente un primer resultado profesional,
- quieres entender cómo funciona la creación de sitios web con IA.

## Qué necesitas antes de empezar

Antes de crear tu sitio web, es útil tener:

- el nombre de tu negocio,
- tu ciudad o zona de servicio,
- el tipo de negocio,
- tus principales servicios o productos,
- un teléfono o correo electrónico de contacto,
- 2-3 imágenes, si las tienes.

No necesitas tenerlo todo perfecto desde el principio. Puedes cambiarlo más tarde en el editor.

## Paso 1 — Abre la aplicación de Centuply

Ve a la aplicación de Centuply:

[Abrir Centuply App](https://app.centuply.com/app)

Inicia sesión en tu cuenta y empieza a crear un nuevo sitio web.

## Paso 2 — Describe tu negocio

En el primer paso, describe el sitio web que quieres crear con palabras simples.

Ejemplo:

> Quiero un sitio web profesional para una peluquería en Pireo. Quiero mostrar servicios, precios, fotos, datos de contacto y un botón para reservar cita.

Puedes escribir de forma natural, como si hablaras con un colaborador.

## Paso 3 — Completa los datos básicos

Según el flujo que utilices, Centuply puede pedirte:

- nombre del negocio,
- categoría del negocio,
- ubicación,
- servicios principales,
- estilo del sitio,
- idioma,
- si quieres formulario, botón de reserva o datos de contacto.

Cuanto más clara sea la información, mejor será el primer resultado.

## Paso 4 — Continúa al pago y la generación

Cuando completes los datos, continúa al paso de pago.

Después de la confirmación, Centuply empieza a crear tu sitio web. La IA prepara:

- páginas,
- textos,
- estructura,
- secciones,
- SEO básico,
- diseño inicial.

## Paso 5 — Revisa el primer resultado

Cuando la generación termine, verás tu sitio web en vista previa.

En esta etapa no tiene que estar todo perfecto. El objetivo es tener una base inicial sólida.

Primero revisa:

- si el nombre del negocio es correcto,
- si los servicios principales aparecen correctamente,
- si la página de inicio tiene sentido,
- si aparecen los datos básicos de contacto,
- si el estilo encaja con el negocio.

## Paso 6 — Haz tus primeros cambios

Después del primer resultado, puedes abrir el editor y mejorar el sitio.

Los cambios iniciales más comunes son:

- cambiar el título de la página de inicio,
- corregir textos,
- añadir o cambiar imágenes,
- cambiar colores,
- añadir una nueva página,
- activar un formulario de contacto,
- revisar el botón de contacto o reserva.

## Paso 7 — Usa el AI Assistant

Dentro del editor puedes pedir cambios al Centuply Assistant.

Ejemplos:

> Haz que la sección principal se vea más premium.

> Escribe un texto más corto para los servicios.

> Haz que el botón de contacto sea más fuerte.

> Sugiere un mejor titular para la página de inicio.

Lo importante es que Centuply primero te muestra una vista previa. Los cambios no se aplican si no haces clic en Apply.

## Paso 8 — Revisa antes de publicar

Antes de publicar, revisa:

- la página de inicio,
- la página de contacto,
- el teléfono y el correo electrónico,
- los enlaces,
- la apariencia en móvil,
- el SEO básico,
- si tienes un dominio o un enlace temporal de vista previa.

También puedes usar Site Health para revisar posibles problemas antes de publicar.

## Paso 9 — Publica el sitio

Cuando esté listo, haz clic en Publish.

Después de publicar, tu sitio web estará disponible online. Si tienes tu propio dominio, puedes conectarlo desde la sección Domains.

## Preguntas frecuentes

### ¿Puedo cambiar el sitio después de generarlo?

Sí. El primer resultado es la base inicial. Puedes cambiar textos, imágenes, colores, páginas y secciones desde el editor.

### ¿Necesito conocimientos técnicos?

No. Centuply está diseñado para que puedas crear y editar un sitio web sin código.

### ¿Necesito tener fotos preparadas?

No necesariamente. Si tienes tus propias fotos, puedes subirlas. Si no, puedes empezar con el resultado inicial y añadirlas más tarde.

### ¿Puedo usar mi propio dominio?

Sí. Después de crear el sitio web, puedes conectar tu dominio desde la sección Domains.

### ¿Qué pasa si no me gusta el primer resultado?

Puedes editarlo con el editor y pedir mejoras al AI Assistant. La gran ventaja de Centuply es que no te quedas limitado al primer resultado.
