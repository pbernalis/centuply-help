# ¡Bienvenido al Centro de Ayuda de Centuply! 🎉

Aquí encontrarás las guías principales que necesitas para crear, editar, gestionar y publicar tu sitio web con IA usando Centuply.

## ¿Qué es Centuply?

Centuply es un **constructor de sitios web con IA** que te ayuda a crear sitios web profesionales con inteligencia artificial.

## Empieza en 5 minutos

1. Abre la aplicación → [app.centuply.com/app](https://app.centuply.com/app)
2. Describe tu negocio con palabras simples
3. Deja que la IA cree tu sitio web
4. Edítalo con el Visual Editor
5. Publícalo cuando esté listo

[→ Ir a la App](https://app.centuply.com/app)
