# Καλώς ήρθες στο Centuply Help Center!

Εδώ θα βρεις **όλες τις απαντήσεις** που χρειάζεσαι για να φτιάξεις, να επεξεργαστείς και να δημοσιεύσεις το AI website σου γρήγορα και εύκολα.

## Τι είναι το Centuply;

Το Centuply είναι ένα **AI Website Builder** που σου επιτρέπει να δημιουργήσεις επαγγελματικά websites με τεχνητή νοημοσύνη σε δευτερόλεπτα — χωρίς γνώσεις κώδικα.

## Γρήγορη Εκκίνηση (5 λεπτά)

1. **Δημιούργησε τον λογαριασμό σου** → [app.centuply.com/app](https://app.centuply.com/app)
2. **Περιέγραψε το website σου** με απλά ελληνικά
3. **Το AI φτιάχνει το site σου** αυτόματα
4. **Επεξεργάσου το** με τον Visual Editor
5. **Δημοσίευσέ το** με ένα κλικ (συμπεριλαμβάνεται custom domain)

Πήγαινε στην αναζήτηση πάνω ή πήγαινε κατευθείαν στο [app.centuply.com/app](https://app.centuply.com/app) και πάτα το κουμπί βοήθειας.

---

**Πώς να το ανεβάσεις:**

1. Πήγαινε στο GitHub repo σου → φάκελος `docs`
2. **Add file** → **Create new file**
3. Όνομα αρχείου: `intro.mdx`
4. Κόπιαρε τον κώδικα από πάνω
5. Commit με μήνυμα: `Add Getting Started intro page`

**Μόλις κάνεις commit & push**, πες μου **“έγινε”** και σου δίνω **αμέσως** τα επόμενα 4-5 άρθρα (AI Builder, Visual Editor, Publishing, Billing) έτοιμα για copy-paste.

Θέλεις να συνεχίσουμε; Πάτα Create new file τώρα! 🚀
