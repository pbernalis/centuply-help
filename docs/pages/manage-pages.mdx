# manage pages

Σε αυτόν τον οδηγό θα μάθεις βήμα-βήμα πώς να χρησιμοποιήσεις αυτή τη λειτουργία στο Centuply.

## Τι θα μάθεις

- Πού βρίσκεται η λειτουργία
- Πότε τη χρησιμοποιείς
- Ποια είναι τα βασικά βήματα
- Τι να προσέξεις πριν κάνεις publish

## Βήματα

1. Άνοιξε το Centuply app.
2. Επίλεξε το site σου.
3. Πήγαινε στη σχετική ενότητα.
4. Κάνε την αλλαγή.
5. Έλεγξε το preview.
6. Πάτησε Save ή Publish όταν είσαι έτοιμος.
