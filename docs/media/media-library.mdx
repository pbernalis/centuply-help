# Media Library: εικόνες, αρχεία και οργάνωση

Η Media Library είναι το σημείο όπου διαχειρίζεσαι τις εικόνες και τα media assets του site σου.

Από εδώ μπορείς να ανεβάσεις φωτογραφίες, να οργανώσεις αρχεία, να ξαναχρησιμοποιήσεις εικόνες σε διαφορετικές σελίδες και να κρατήσεις το περιεχόμενο του site σου τακτοποιημένο.

## Πότε χρησιμοποιείς τη Media Library

Χρησιμοποίησε τη Media Library όταν θέλεις να:

- ανεβάσεις εικόνες της επιχείρησης,
- αλλάξεις εικόνες σε sections,
- οργανώσεις φωτογραφίες σε folders,
- βρεις εικόνες που έχουν ήδη ανέβει,
- ξαναχρησιμοποιήσεις εικόνες σε άλλη σελίδα,
- ελέγξεις ποια assets χρησιμοποιούνται,
- καθαρίσεις παλιά ή άχρηστα media.

## Τι μπορείς να αποθηκεύσεις

Ανάλογα με τις δυνατότητες του site σου, η Media Library μπορεί να περιλαμβάνει:

- εικόνες,
- logo files,
- icons,
- videos ή video links,
- documents,
- generated assets,
- external media URLs.

Οι εικόνες είναι συνήθως το πιο σημαντικό asset για ένα επαγγελματικό business website.

## Βήμα 1 — Άνοιξε τη Media Library

Μπες στο Centuply app, επίλεξε το site σου και άνοιξε την ενότητα **Media** ή **Media Library**.

Θα δεις τα assets που έχουν ήδη ανέβει ή χρησιμοποιούνται στο site.

## Βήμα 2 — Ανέβασε νέα εικόνα

Πάτησε **Upload** ή σύρε εικόνες μέσα στη Media Library.

Πριν ανεβάσεις, έλεγξε ότι οι εικόνες:

- είναι καθαρές,
- έχουν καλή ανάλυση,
- δεν είναι υπερβολικά σκοτεινές,
- δεν είναι πολύ μεγάλες χωρίς λόγο,
- ταιριάζουν με το ύφος του site.

## Βήμα 3 — Οργάνωσε τα media

Αν έχεις πολλές εικόνες, οργάνωσέ τες με folders ή tags.

Παραδείγματα οργάνωσης:

- Hero images
- Services
- Team
- Products
- Gallery
- Logo
- Backgrounds
- Social

Η σωστή οργάνωση βοηθά πολύ όταν το site μεγαλώσει.

## Βήμα 4 — Χρησιμοποίησε εικόνα σε section

Όταν επεξεργάζεσαι ένα section στον Visual Editor, μπορείς να επιλέξεις εικόνα από τη Media Library.

Παραδείγματα:

- Hero image στην αρχική σελίδα,
- εικόνα υπηρεσίας,
- φωτογραφία ομάδας,
- εικόνα gallery,
- background εικόνα,
- logo στο header.

## Βήμα 5 — Έλεγξε πού χρησιμοποιείται ένα asset

Πριν διαγράψεις μια εικόνα, έλεγξε αν χρησιμοποιείται σε κάποια σελίδα ή section.

Αν διαγράψεις εικόνα που χρησιμοποιείται, μπορεί να χαλάσει η εμφάνιση μιας σελίδας.

Καλύτερα πρώτα να την αντικαταστήσεις και μετά να τη διαγράψεις.

## Βήμα 6 — Καθάρισε παλιά ή άχρηστα assets

Με τον καιρό μπορεί να μαζευτούν εικόνες που δεν χρειάζεσαι.

Μπορείς να καθαρίσεις:

- παλιές εικόνες,
- διπλότυπα,
- προσωρινά uploads,
- εικόνες που δεν χρησιμοποιούνται,
- assets που δεν ταιριάζουν πλέον στο brand.

Μην διαγράφεις βιαστικά αρχεία αν δεν είσαι σίγουρος πού χρησιμοποιούνται.

## Βήμα 7 — Έλεγξε mobile εμφάνιση

Μια εικόνα μπορεί να φαίνεται ωραία σε desktop αλλά όχι σε κινητό.

Μετά την αλλαγή εικόνας, έλεγξε:

- αν κόβεται σωστά,
- αν το πρόσωπο ή το προϊόν φαίνεται,
- αν το κείμενο παραμένει ευανάγνωστο,
- αν το section φαίνεται ισορροπημένο.

## Καλές πρακτικές για εικόνες

- Προτίμησε πραγματικές φωτογραφίες της επιχείρησης.
- Χρησιμοποίησε φωτεινές και καθαρές εικόνες.
- Απόφυγε θολές φωτογραφίες.
- Μην ανεβάζεις πολλές εκδοχές της ίδιας εικόνας χωρίς λόγο.
- Κράτα συνεπές ύφος.
- Έλεγξε πάντα πώς φαίνεται η εικόνα σε κινητό.
- Μην χρησιμοποιείς εικόνες χωρίς δικαίωμα χρήσης.

## Συχνές ερωτήσεις

### Μπορώ να ανεβάσω δικές μου φωτογραφίες;

Ναι. Μπορείς να ανεβάσεις φωτογραφίες από τον υπολογιστή σου και να τις χρησιμοποιήσεις στο site.

### Πρέπει να οργανώνω τις εικόνες σε folders;

Δεν είναι υποχρεωτικό, αλλά βοηθά πολύ όταν έχεις πολλά assets.

### Μπορώ να χρησιμοποιήσω την ίδια εικόνα σε πολλές σελίδες;

Ναι. Μια εικόνα μπορεί να χρησιμοποιηθεί σε περισσότερα από ένα σημεία.

### Τι γίνεται αν διαγράψω εικόνα που χρησιμοποιείται;

Μπορεί να λείπει από τη σελίδα ή να χαλάσει η εμφάνιση. Έλεγξε πρώτα πού χρησιμοποιείται.

### Τι εικόνες να προτιμώ;

Προτίμησε καθαρές, φωτεινές και πραγματικές εικόνες που δείχνουν την επιχείρηση, την ομάδα, τον χώρο ή τα προϊόντα.
