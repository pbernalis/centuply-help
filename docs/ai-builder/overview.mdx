# AI Website Builder

Το **AI Website Builder** είναι η πιο ισχυρή λειτουργία του Centuply.

Με λίγα λόγια στα ελληνικά, το AI δημιουργεί ολόκληρο το website σου αυτόματα — σχεδιασμό, κείμενα, εικόνες και δομή.

## Πώς δουλεύει βήμα-βήμα

1. **Περιγράψτε την ιδέα σας**  
   Γράψτε τι ακριβώς θέλετε (π.χ. «εστιατόριο ιταλικής κουζίνας στην Αθήνα με online παραγγελίες»).

2. **Το AI δημιουργεί το site**  
   Σε λίγα δευτερόλεπτα παράγει πλήρες website με:
   - Σύγχρονο και responsive design
   - SEO-friendly κείμενα
   - Υψηλής ποιότητας εικόνες
   - Έτοιμες σελίδες (Home, Menu, Contact, κλπ.)

3. **Επεξεργαστείτε με τον Visual Editor**  
   Αλλάξτε οτιδήποτε θέλετε με drag & drop — χωρίς κώδικα.

4. **Δημοσιεύστε**  
   Με ένα κλικ το site σας είναι live στο δικό σας domain.

## Κύρια πλεονεκτήματα

- Πολύ γρήγορη δημιουργία (δευτερόλεπτα αντί για μέρες)
- Επαγγελματικό αποτέλεσμα χωρίς σχεδιαστή
- Εύκολη επεξεργασία από οποιονδήποτε
- Συμπεριλαμβάνεται hosting + SSL
